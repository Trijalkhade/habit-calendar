Habit Calendar

Install on your system in seconds — no security warnings

✔ No security warnings ✔ Auto-updates ✔ 1 command
⚡ Install via Homebrew
  1. Open the Terminal app (press ⌘ + Space, type "Terminal", hit Enter).
  2. Paste the command below and press Enter.
  3. The app will be installed and ready to use — zero warnings!
Terminal
brew install --cask trijalkhade/tap/habit-calendar
✔ No security warnings ✔ No Homebrew needed
⚡ Install via Script
  1. Open the Terminal app (press ⌘ + Space, type "Terminal", hit Enter).
  2. Paste the command below and press Enter.
  3. The script downloads, installs, and opens the app automatically.
Terminal
curl -sSL https://raw.githubusercontent.com/Trijalkhade/habit-calendar/main/install.sh | bash
📥 Manual Installation
  1. Download the .dmg file from GitHub Releases.
  2. Open the .dmg and drag the app to your Applications folder.
  3. Right-click the app in Applications → select Open → click Open again.
  4. Or run in Terminal: xattr -cr "/Applications/Habit Calendar.app"
✔ No SmartScreen warnings ✔ Silent install ✔ Auto-launches
⚡ Install via PowerShell
  1. Press Win + X and select Terminal (or search for PowerShell in the Start menu).
  2. Paste the command below and press Enter.
  3. The script downloads, installs, and opens the app automatically.
PowerShell
iwr -useb https://raw.githubusercontent.com/Trijalkhade/habit-calendar/main/install.ps1 | iex
📥 Manual Installation
  1. Download the .exe or .msi file from GitHub Releases.
  2. If a Windows protected your PC warning appears, click More info.
  3. Then click Run anyway.
📦 Install via .deb package
  1. Download the .deb file from GitHub Releases.
  2. Run the command below (replace the filename if needed).
Bash
sudo dpkg -i Habit.Calendar_*_amd64.deb
📦 Run as AppImage
  1. Download the .AppImage file from GitHub Releases.
  2. Make it executable and run it.
Bash
chmod +x Habit.Calendar_*_amd64.AppImage && ./Habit.Calendar_*_amd64.AppImage
🔧 Build from Source
  1. Make sure you have Node.js, Rust, and Tauri's system dependencies installed.
  2. Run the command below to clone, build, and install.
Bash
git clone https://github.com/Trijalkhade/habit-calendar.git && cd habit-calendar && npm install && npm run tauri build
