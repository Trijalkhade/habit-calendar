Habit Calendar

Install Habit Calendar on your system in seconds

⚡ 1-Click Install (Bypasses Gatekeeper warnings)
  1. Open the Terminal app on your Mac.
  2. Copy and paste the command below, then press Enter.
  3. The script will download, install, and launch the app automatically.
Terminal
curl -sSL https://raw.githubusercontent.com/Trijalkhade/habit-calendar/main/install.sh | bash
⚡ 1-Click Install (Bypasses SmartScreen warnings)
  1. Search for PowerShell in your Start menu.
  2. Copy and paste the command below, then press Enter.
  3. The script will download, install, and open Habit Calendar.
PowerShell
iwr -useb https://raw.githubusercontent.com/Trijalkhade/habit-calendar/main/install.ps1 | iex
⚡ Build from Source / Manual Installation
  1. Tauri binaries are compiled for Windows (.msi/.exe) and macOS (.dmg) by default.
  2. For Linux, you can clone and build the app using npm run tauri build.
Bash
git clone https://github.com/Trijalkhade/habit-calendar.git && cd habit-calendar && npm install && npm run tauri build
